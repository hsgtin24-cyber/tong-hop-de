Bài 1: NĂNG LƯỢNG

2024-2025 OLYMPIC 10 HCM

6 Điểm Time Limit: 1.0s Memory: 256MB

0. Đề bài chi tiết

An có quỹ thời gian N ngày để luyện tập cho kì thi. Do đó, An phải lập thời khóa biểu cho N buổi tối mà trong mỗi buổi đó An sẽ luyện tập hoặc ngủ sớm. Mỗi tối, nếu An ngủ sớm thì năng lượng sẽ tăng lên A đơn vị, nếu An luyện tập thì năng lượng sẽ giảm B đơn vị.

An muốn luyện tập nhiều nhất có thể và cũng muốn duy trì mức năng lượng như trước khi bắt đầu luyện tập, nghĩa là mức năng lượng sẽ không giảm và cũng không tăng. An nhận định rằng muốn duy trì năng lượng thì tổng độ giảm phải bằng tổng độ tăng.

Yêu cầu:

Hãy viết chương trình giúp An lập thời khóa biểu thỏa điều kiện trên.

Dữ liệu vào (Input) từ file NANGLUONG.INP:

  • Dòng đầu tiên ghi số nguyên dương N (1 ≤ N ≤ 106).
  • Dòng thứ hai ghi số nguyên dương A (1 ≤ A ≤ 103).
  • Dòng thứ ba ghi số nguyên dương B (1 ≤ B ≤ 103).

Kết quả (Output) ghi ra file NANGLUONG.OUT:

Một số nguyên là số ngày An sẽ luyện tập. Nếu An không thể lập được thời khóa biểu theo yêu cầu trên thì ghi -1.

Ví dụ (Sample Test)

1. Phân tích Toán học & Logic

Dạng bài: Phương trình Toán học bậc nhất

Bài toán yêu cầu tìm số ngày luyện tập lớn nhất sao cho năng lượng cân bằng. Ta có thể mô hình hóa bài toán bằng các biến đại số:

  • Gọi x là số ngày luyện tập (x ≥ 0).
  • Gọi y là số ngày ngủ sớm (y ≥ 0).
  • Tổng số ngày: x + y = N ⇒ y = N - x   (1)
  • Tổng năng lượng giảm: x × B
  • Tổng năng lượng tăng: y × A

Biến đổi đại số

Theo yêu cầu duy trì năng lượng: Tổng giảm = Tổng tăng.

$$ x \cdot B = y \cdot A $$ $$ \Leftrightarrow x \cdot B = (N - x) \cdot A $$ $$ \Leftrightarrow x \cdot B = N \cdot A - x \cdot A $$ $$ \Leftrightarrow x \cdot (A + B) = N \cdot A $$
$$ \Rightarrow x = \frac{N \cdot A}{A + B} $$

Cảnh báo Giới hạn & Kiểu dữ liệu

- Với giới hạn N ≤ 106, A, B ≤ 103, tử số N × A tối đa là 109.

- Trong C++, tích này nằm trong giới hạn của số nguyên 32-bit có dấu (int chứa được đến 2 × 109). Tuy nhiên, để an toàn 100%, nên dùng long long.

- Trong Python, số nguyên tự động mở rộng không giới hạn, nên không lo tràn số.

- Do x đại diện cho số ngày luyện tập nên x bắt buộc phải là số nguyên. Nghĩa là (N × A) phải chia hết cho (A + B).

2. Phân tích Cách giải

Cách 1 Vét cạn (Brute Force) - Cho Subtask nhỏ

Ý tưởng: Vì An muốn số ngày luyện tập x là lớn nhất có thể, ta có thể thử (vét cạn) giá trị của x giảm dần từ N xuống 0. Tại mỗi giá trị x, ta tính số ngày ngủ y = N - x, sau đó kiểm tra điều kiện x × B == y × A. Giá trị đầu tiên thỏa mãn chính là đáp án. Nếu duyệt hết mà không có, trả về -1.

// Mã giả thuật toán Vét cạn
FOR x FROM N DOWN TO 0:
    y = N - x
    IF x * B == y * A THEN:
        RETURN x
RETURN -1

Code minh họa (Python):

def solve_brute_force(n, a, b):
    # Duyệt x từ n giảm dần về 0
    for x in range(n, -1, -1):
        y = n - x
        if x * b == y * a:
            return x
    return -1

Cách 2 Công thức Toán học (Tối ưu) - Cho Mọi Subtask

Ý tưởng: Dựa vào chứng minh toán học ở Phần 1, ta đã có công thức trực tiếp: x = (N × A) / (A + B). Vì x đại diện cho số ngày luyện tập (phải là số nguyên), ta chỉ việc kiểm tra xem phép chia này có dư hay không. Nếu dư 0, kết quả là thương của phép chia. Ngược lại, không tồn tại phương án (in ra -1).

// Mã giả thuật toán Tối ưu
tuSo = N * A
mauSo = A + B
IF tuSo % mauSo == 0 THEN:
    RETURN tuSo / mauSo
ELSE:
    RETURN -1

Code Python Chuẩn thi HSG:

import sys

# Tên file theo yêu cầu đề bài
INPUT_FILE = "NANGLUONG.INP"
OUTPUT_FILE = "NANGLUONG.OUT"

def nhap():
    try:
        with open(INPUT_FILE, "r") as f:
            data = f.read().split()
            if len(data) < 3: return None
            return int(data[0]), int(data[1]), int(data[2])
    except:
        return None

def xuli(n, a, b):
    # x * b = (n - x) * a => x * (a + b) = n * a
    tu_so = n * a
    mau_so = a + b
    
    # Kiểm tra điều kiện chia hết để x là số nguyên
    if tu_so % mau_so == 0:
        return tu_so // mau_so
    return -1

def xuat(result):
    with open(OUTPUT_FILE, "w") as f:
        f.write(str(result))

# Main logic
data = nhap()
if data:
    n, a, b = data
    xuat(xuli(n, a, b))
